# Problem 5: A Crude Server

A clean, light-weight CRUD backend service built with ExpressJS, TypeScript, and SQLite for data persistence.

## Features

- Full CRUD interfaces for a `Book` resource.
- Dynamic filtering capabilities (filter books by title, author, or genre).
- Persistent storage using SQLite (no setup required from the reviewer side).
- Strongly-typed request/response architecture.

## Installation & Setup

1. Open your terminal in this directory (`problem5`).
2. Install the necessary dependencies:

```bash
npm install
```
