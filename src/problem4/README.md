# Problem 4: Three Ways to Sum to N

## File Structure

- `sum_to_n.ts`: Contains the 3 required implementation functions (`sum_to_n_a`, `sum_to_n_b`, `sum_to_n_c`) accompanied by JSDoc comments describing their Time and Space complexities.
- `sum_to_n.test.ts`: Contains the automated test cases verifying correctness, boundary conditions ($n = 0$), and negative inputs for all three implementations.

## How to Run the Tests

Ensure you have Node.js (v18+) installed, then open your terminal in this directory and execute:

```
npx tsx sum_to_n.test.ts
```
